DOM COMPLETE NOTES
The Document Object Model (DOM) is a structured way for web browsers to represent and interact with HTML pages, treating every part of a page (like tags, text, and attributes) as objects.(It's how JavaScript “talks” to HTML.)
The DOM is explained as a model that represents the structure of an HTML document as a tree of objects (head, body, p, img ). Every element, attribute, and text is considered a "node" in this tree

    Example HTML:
    <html>
    <body>
        <h1>Hello</h1>
        <p>Paragraph</p>
    </body>
    </html>

    Corresponding DOM Tree:

    Document
    └── html
        └── body
            ├── h1
            │    └── "Hello"
            └── p
                └── "Paragraph"

1.Accessing DOM elements:-
let el = document.getElementById("myId");

    let el = document.getElementsByTagName("p"); // HTMLCollection
    let el = document.querySelector(".myClass"); // First match

2.Changing Content
    el.innerHTML = "<strong>Bold Text</strong>";
    el.textContent = "Plain text only";
    el.innerText = "Visible text only";

3.Changing Attributes
    el.setAttribute("src", "image.jpg"); //it overwrites
    let src = el.getAttribute("src");
    el.id = "newId";
    src = "image.png";

4.Changing Styles
    Inline style:
        el.style.color = "red";
        el.style.backgroundColor = "yellow";
    Using CSS classes:
        el.classList.add("active");
        el.classList.remove("hidden");
        el.classList.toggle("dark-mode");

5.Traversing the DOM (code details in two.html)
