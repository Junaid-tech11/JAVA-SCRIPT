# for loop

- it is good for the creation of arrays and objects
- it is not good for iterating over arrays and objects

# for-of-loop

- it is good for iterating over arrays and objects and find index of an element in an array
- it is not good for the creation of arrays and objects

# for-in-loop

- it is good for iterating over objects and find index of an element in an array
- it is not good for the creation of arrays and objects

# forEach-loop

- it is good for iterating over arrays and objects and find index of an element in an array
- it is not good for the creation of arrays and objects
- it not works with break and continue statements
- it also not compatible with async/await functions
